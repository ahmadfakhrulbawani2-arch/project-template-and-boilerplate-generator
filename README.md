## Description

**PROJECT-TEMPLATE-AND-BOILERPLATE-GENERATOR** is an experimental shell scripts to automate project setup especially vanilla project (without using library or framework). My intention is to sharpen my shell scripting skill and automated architecture.

## How to Use

**Make sure you have this:**

- Git Bash
- Bash Terminal (optional)
- VSCode (optional)

**Step-by-step:**

- Prepare a folder or directory to group your auto generated project. Enter that folder by:

```bash
cd /to-your-folder
```

- Clone this repository with:

```bash
git clone https://github.com/ahmadfakhrulbawani2-arch/project-template-and-boilerplate-generator .
```

- In this directory, open Bash Terminal (or with Git Bash) then try to run a script. For example

```bash
chmod +x start-web.sh
./start-web.sh
```

- Enter the information needed.
- All your activity will be recorded automatically in [history.log](history.log).
